# مطعم الزعيم - قائمة الطعام الإلكترونية

![مطعم الزعيم](https://i.imgur.com/XYZ123.png)

تطبيق ويب تفاعلي لعرض قائمة مطعم الزعيم للمأكولات الشعبية. يتيح التطبيق للمستخدمين استعراض الأطباق المختلفة بواجهة سهلة الاستخدام ومتجاوبة مع جميع أحجام الشاشات.

## المميزات

- واجهة مستخدم متجاوبة تعمل على جميع أحجام الشاشات
- دعم اللغتين العربية والإنجليزية
- تصنيف الأطباق إلى فئات (علب وأصناف، أصناف جانبية، سندوتشات)
- عرض تفاصيل كل طبق مع السعر والوصف
- تصميم عصري بألوان مميزة
- أيقونات معبرة لكل نوع من أنواع الطعام

## التقنيات المستخدمة

- React.js
- Vite
- Tailwind CSS
- Font Awesome

## متطلبات التشغيل

- Node.js (الإصدار 14 أو أحدث)
- npm أو yarn

## طريقة التثبيت والتشغيل

1. استنساخ المشروع:
   ```bash
   git clone https://github.com/yourusername/zaeem-restaurant.git
   cd zaeem-restaurant
   ```

2. تثبيت الاعتماديات:
   ```bash
   npm install
   # أو
   yarn
   ```

3. تشغيل المشروع محلياً:
   ```bash
   npm run dev
   # أو
   yarn dev
   ```

4. فتح المتصفح على العنوان: `http://localhost:5173`
